##BRANCHES
- branches are used in cases when multiple developers are working on adding features for a project parallely.

  *git branch* - to check branch <br>
  *git branch -M main* - to rename branch <br>
  *git checkout <-branch name->* - to navigate <br>
  *git checkout -b <-new branch name->* - to create new branch <br>
  *git branch -d <-branch name->* - to delete a branch <br>
